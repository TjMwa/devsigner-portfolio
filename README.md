## 🚀 Overview

A high-performance, minimalist portfolio designed for **Tyson Mwangi**, a Nairobi-based Product Architect and Engineer. This project showcases the intersection of aesthetic product design and robust backend engineering, featuring a cyberpunk-inspired aesthetic with state-of-the-art animations.

> [!NOTE]

---

## 🛠️ Technology Stack

### Core Engine
- **Frontend**: [React 19](https://react.dev/) (Strict Mode), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Custom Design System)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Vite**: Ultra-fast module bundling

### Design Systems
- **Palette**: Slate-900 Dark Mode with Lime-400 Accents.
- **Typography**: Inter (Body) & Space Grotesk (Display).
- **Interactive**: Matrix digital rain background, custom precision cursor.

1. **Matrix Digital Rain**: An interactive, low-opacity background element providing a subtle cyberpunk vibe without sacrificing readability.
2. **Custom Precision Cursor**: A custom React-managed cursor that reacts to hover states and provides tactile visual feedback.
3. **Responsive Grid Architecture**: Seamless transitions from HD desktops to mobile screens.
4. **Data-Driven Sections**:

   - **Experience**: Detailed timeline of engineering leadership at InnovateTech and DataSphere.

## 🏗️ Architecture

The codebase follows the **Single Source of Truth (SSOT)** principle:
- `constants.tsx`: Centralized repository for all portfolio data.
- `components/`: Modular, reusable UI components.
- `types.ts`: Domain-specific TypeScript interfaces ensuring type safety.

## 🖥️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS)
- NPM or PNPM

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tysonmwangi/devsigner-portfolio.git
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Development Mode**
   ```bash
   npm run dev
   ```

4. **Production Build**
   ```bash
   npm run build
   ```
